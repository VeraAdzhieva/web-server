# Веб-сервер

## Разработка веб-сервера:
- Обработка HTTP-методов GET .
- Обработка кодов состояния.
- Поддержка MIME-типов и заголовков.
- Парсинг URL: поддержка пробелов и query-параметров.

Для разработки использованы модуль socketserver, архитектура Threading, стандартные библиотеки os, sys, threading, datetime.
Проведено тестирование unit, а также нагрузочное тестирование (ab).

## Результаты нагрузочного тестирования.
```ab.exe -n 50000 -c 100 http://127.0.0.1:8080/httptest/wikipedia_russia.html```

### Параметры нагрузки: 
- Всего запросов: 50000
- Параллельные соединения(Concurrency): 100
- Объем переданных данных: 954824 bytes

## Ключевые метрики:
- Requests per second: 508.92
- Среднее время ответа (Time per request): 1.965 мс
- Пропускная способность (Transfer rate): 474,614 КБ/сек
- Ошибки: 0
